## 📌 Problem Statement

In many student grant and funding systems:

- ❌ Lack of transparency in fund usage  
- ❌ Manual milestone verification  
- ❌ Delayed or misused fund disbursement  
- ❌ No real-time tracking for sponsors  

Sponsors cannot easily ensure that funds are released only when real progress is made.

---

## 💡 Our Solution

**ChainGrant** is a blockchain-powered milestone-based escrow system built on **Algorand** that:

✅ Locks sponsor funds in a smart contract  
✅ Releases funds automatically on milestone approval  
✅ Provides real-time transparent tracking  
✅ Eliminates manual intervention  
✅ Ensures trust between sponsor and student  

---

## 🧠 Key Features

- 🔐 Smart contract escrow on Algorand  
- 📊 Milestone progress tracking  
- 💸 Automatic ALGO disbursement  
- 🧾 On-chain transaction history  
- 🗳️ DAO-style approval flow  
- ⚡ Fast and low-cost TestNet deployment
